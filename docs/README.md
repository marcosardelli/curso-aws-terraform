# README

Descripción

* Repositorio de apoyo para el curso de AWS con Terraform.
* Contiene ejemplos, módulos y ejercicios prácticos para desplegar infraestructura en AWS.

Requisitos

* Cuenta de AWS con permisos adecuados.
* Terraform (v1.x recomendado).
* AWS CLI configurado (credenciales en \~/.aws/credentials).
* Git

Estructura

* docs/ -- Material del curso y guías
* examples/ -- Ejemplos prácticos por temática
* modules/ -- Módulos reutilizables de Terraform
* envs/ -- Carpetas por entorno (dev, staging, prod)
* README.md -- Este archivo
